  • Linguagens de Programação ou Desenvolvimento
  • HTML

    Mexer com HTML sem dúvidas é uma das coisas mais gostosas e divertidas de se fazer. Meu primeiro contato com essa técnologia foi lá em 2010, quando os blogs estavam vivendo seu ápice, e quem não queria ter uma página personalizada? Depois disso, honestamente, não tive muito contato até este ano de 2020.

    Voltei a ter contato com ela no curso técnico, para me aprofundar, fiz o Curso em Vídeo de HTML do Guanabara, depois fiz o curso do Estudonauta e o da Fundação Bradesco. E finalmente, quando estava mais confortável, estudei diretamente a documentação fornecida pela W3C.

    Entendo que a função da HTML é essencialmente semântica, por isso tento utilizar as tag da forma mais correta possível, pois compreendo a importante disso não apenas para um bom rankeamento em mecanismos de busca, como para a acessibilidade de pessoas com deficiência.

  • CSS

    Juntamente com HTML, as CSS são divertidíssimas. Muitas vezes passo horas simplesmente criando uma página de teste para experimentar estilos e funções. Minha trajetórias com as CSS foi paralela com a HTML. A diferença é que neste ano conheci o milagre chamado "frameworks"!

    Por enquanto estou familirizado com o W3 e com o Materialize.

  • JavaScript

    O JavaScript, honestamente, foi um choque em um primeiro momento 😄

    A primeira linguagem de programação que aprendi foi Java, então quando fui para o JavaScript eu ficava "mas como assim essa variável que era um int virou uma string?". Apesar de a lógica de programação ser a mesma, a cada linguagem temos que nos adaptar a seu estilo próprio.

    Atualmente gosto muito de trabalhar com JS. Vi um pouco sobre ela no curso técnico, fiz o curso no Estudonauta e na Fundação Bradesco, que foram muito importantes para eu me familiarizam com os fundamentos da linguagem. Mas o que realmente me ajudou a conhecer esta linguagem de forma mais profunda foi o Bootcamp de FullStack que fiz pelo IGTI.

    Ali me aprofundei muito no conceito e funcionamento de funções. Tive contato com o Node.js e MongoDB(back-end), além de ter aprendido sobre APIs e uso de arquivos JSON.

    Quanto aos frameworks, estou mais familiarizado com o JQuery, mas também já tive contato com o React.

    Mas na dúvida, o site da Mozila está aí para isso!

  • Java

    Java foi a primeira linguagem de programação que aprendi. E apesar de muitas pessoas falarem mal dela, eu tenho um grande carinho, e gosto bastante de criar aplicativos desktop utilizando Java, só por diversão.

    Por ter me identificado muito, busquei aprender o máximo que podia, fiz no curso em vídeo Java Básico, e POO em Java, na Fundação Bradesco fiz o curso de Java Básico e Java Avançado. E comecei a estudar diretamente a documentação. Além de ter arriscado um pouco utilizar o Android Studio, mas infelizmente meu notebook não é tão potente e meu processador quase derreteu! 🤣

    Claro que comecei a estudar programação este ano, e nunca trabalhei na área, e me falta muita experiência e tenho muito a aprender. Mas pelo menos já me sinto muito confortável com conceitos como: Classes, Objetos, Encapsulamento, Polimorfismo, Herança, etc.
